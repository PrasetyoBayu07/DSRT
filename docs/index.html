Overview

DSRT is a modular 2D simulation engine built with HTML5 Canvas and ES6 modules. This documentation explains its core structure, available modules, and API reference.

Core Concepts

  • Core: Canvas, Boolean, and Flow systems
  • UI: Buttons, sliders, input fields, and drag zones
  • Graphics: Drawing utilities, charts, and text rendering
  • Physics: Particles, forces, and mechanical systems
  • Lab: Experimental components like beaker, burner, thermometer
