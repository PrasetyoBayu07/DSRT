Overview

DSRT is a modular 2D simulation engine built with HTML5 Canvas and ES6 modules. This documentation covers its structure, available components, and APIs.

Core Concepts

  • Core: Canvas, Boolean, Flow systems
  • UI: Buttons, sliders, inputs, drag zones
  • Graphics: Drawing utilities & text rendering
  • Physics: Particles, forces, and mechanical simulation
  • Lab: Experimental elements like beaker, burner, thermometer
