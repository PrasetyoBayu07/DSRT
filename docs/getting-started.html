Installation

git clone https://github.com/PrasetyoBayu07/DSRT.git
cd DSRT
npm install

Run Locally

npm start

Importing DSRT

import { DSRT } from './src/dsrt.js';

Once imported, you can instantly access modules like DSRT.Canvas, DSRT.Graphics, and DSRT.UI for interactive experiments.
